Explore the Pokemon world and catch/inspect Pokemon!
# Commands:
* help - Displays a help message
* exit - Exits the Pokedex
* map - Shows the first 20 locations in a response, and the next 20 on each subsequent call
* mapb - Shows the previous page of 20 locations or informs you that you are on the first page of results
* explore - Shows which pokemon can be encountered in a user-provided location
* catch - Try to catch a user-provided Pokemon
* inspect - Print the name, height, weight, stats, and type(s) of the user-provided Pokemon
* pokedex - List the names of all caught Pokemon
